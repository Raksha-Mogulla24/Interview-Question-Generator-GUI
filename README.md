# Interview Question Generator (Python GUI)

A Python GUI application that generates technical interview questions based on selected topic and difficulty level.

## Features

* Simple and user-friendly GUI
* Select interview **topic**
* Choose **difficulty level (Easy, Medium, Hard)**
* Generate **3 questions or 100 questions instantly**
* Saves generated questions to a text file
* Useful for **students preparing for technical interviews**

## Technologies Used

* Python
* Tkinter (GUI)
* Random module

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/interview-question-generator.git
```

2. Navigate to the project folder

```
cd interview-question-generator
```

3. Run the program

```
python interview_question_generator_gui.py
```
